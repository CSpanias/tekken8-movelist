Attack

Flash Punch (1,1,2) connection:

  1. Set CPU on Random Standing Block
  2. Press 2 only when 1,1 connects
    3. for 3' on each side
  3. It's better to be correct and slow, i.e., press late/completely miss 2 when 1,1 connects, rather that wrong and fast, i.e., press 2 when 1,1 doesn't connect
  4. Tip: Buffer the second 1 and be ready for 2 by placing your thumb upon it

Defence

Lows

  1. Level 1: ✅
    • Pick Bryan (he has the slowest snake edge of the game): A1 ➔ df,3 (M40,29f), A2 ➔ b,1 (M51,20f)
    • Block both movements successfully. Focus on animation as both movements have identical sound.
  2. Level 2:
    • Pick Dragunov (slightly faster snake edge vs. Bryan): A1 ➔ df,3 (M48,27f), A2 ➔ db,3,1+2 (M49,19f)
    • A2 is a snake edge variation which fakes it and ends in a grab. The moment the movement animation starts, duck and if grab comes punish.

Throws

Tip: Try looking at the shoulders and overall character profile rather than zeroing in on the hands!

  1. Pick Jun and record three actions: A1 ➔ 1,1 (move 15), A2 ➔ 2+4 (generic throw, move 119), A3 ➔ uf,1+2 (command throw, move 124)
  2. Set Action Intervals ➔ Shuffle
  3. Level 1:
    • Activate A2, break it 10 times in a row from both sides. Repeat for A3.
    • Activate A1 & A2: A1 ➔ block, A2 ➔ break. Repeat for A1 & A3.
    • Goal: Break 10 throws in each drill without never pressing throw break vs. A1
  4. Level 2:
    • Activate A2 & A3 ➔ try to break as many throws as possible for 3'
    • Activate A1, A2, & A3: A1 ➔ try to break as many throws as possible without never pressing throw break vs. A1 for 3'

Punishments

Important: If the Punish word on the Attack Display information doesn't turn yellow, it's not a punish!

Block Punishment

  1. Pick Jun as opponent and go to Punishment Training
  2. Tip: Set Attack Warnings to OFF for all moves
  3. Level 1:
    • Pick Shunkei (M1,-12oB,M) ➔ punish it with b,1,2 5 times from each side
    • Pick Kanamegaeshi (M11,-37oB,HL) ➔ Repeat as above with WS,2
    • Pick Ame-no-Nuboko (M15,-20oB,M) ➔ Repeat as above with f,n,df,2
  4. Level 2:
    • Record A1 ➔ Shunkei (M24), A2 ➔ db,4 (-11oB,L), A3 ➔ Ame-no-Nuboko (M115)
    • Activate A2 & A3, block and punish accordingly for 3' each side: A3 ➔ f,n,df,2, A2 ➔ WS,4,4
  5. Level 3:
    • Activate all three and punish accordingly for 3' each side

Whiff Punishment

  1. Pick Jun as opponent.
  2. Practice settings → Training Mode: Defense
  3. Level 1:
    • Practice settings → Action Intervals: Normal
    • A1 → Kazakiri (M43,df,3)
    • Goal: Make space for Jun to whiff and launch punish (f,F,3). Focus on the range of your whiff punish.
  4. Level 2:
    • Practice settings → Action Intervals: Shuffle
    • A2 → Mizuho (M41,df,2)
    • Goal: Same as Level 1, but punish with f,n,df,2
  5. Level 3:
    • Practice settings → Action Intervals: Short
    • A1 → forward dash,df,3, A2 → Mizuho (M41,df,2)
    • A1 → Block, A2 → Whiff punish
    • Tip: Backdash continously while keeping the right space to whiff punish
  6. Level 4:
    • Practice settings → Action Intervals: Short
    • A1 → forward dash,df,3, A2 → Mizuho (M41,df,2), A3 → Katabami (M44, df,4)
    • A1 → Block, A2 → Whiff punish, A3 → Block punish (b,1,2)

While Standing Punishment

  1. Pick Bryan as opponent
  2. Level 1:
    • A1 ➔ db,4 (unseeable), A2 ➔ df,3 (seeable), A3 ➔ d,df,f,3
    • A1 ➔ block, A2 ➔ ws,2, A3 ➔ ws,1,2
  3. Level 2:
    • A1 ➔ 1,n,db,3, A2 ➔ 1,n,df,3, A3 ➔ 1,n,uf,4, A4 ➔ 1,n,f,4,1
    • A1 ➔ take it, A2 ➔ ws,2, A3 ➔ stand block, A4 ➔ ws,4,4
